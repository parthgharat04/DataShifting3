Data Shifting Removal Help

About Data Shifting

Data shifting occurs when newlines are inserted within text-qualified fields, causing the data to shift and become misaligned in subsequent rows.

How This Tool Works

This application fixes data shifting by:

  • Detecting and handling embedded quotes in text fields
  • Properly counting columns to identify field boundaries
  • Cleaning content by replacing newlines within qualified fields
  • Generating detailed error logs for problematic rows

How to Use

  1. Upload your file using the file uploader
  2. Select the appropriate delimiter (default is |^|)
  3. Choose the text qualifier (default is ")
  4. Click "Process File" to start the correction process
  5. Download the corrected file and review any error logs
